Accept payments via Echezona Payment Gateway in your WooCommerce store.

== Description ==

Echezona Payment Gateway for WooCommerce is a WooCommerce payment gateway plugin that allows you to accept payments through the Echezona Payment Gateway. This plugin enables your WooCommerce store to process payments securely and efficiently.

= Features =

* Accept payments via Echezona Payment Gateway
* Support for multiple currencies (NGN, USD, GBP, EUR, GHS, KES, UGX, ZAR)
* Test mode for development and testing
* Support for saved cards
* Split payment functionality
* Custom metadata support
* Automatic order completion
* WooCommerce Blocks support
* High-Performance Order Storage (HPOS) compatible

= Supported Currencies =

* Nigerian Naira (NGN)
* US Dollar (USD)
* British Pound (GBP)
* Euro (EUR)
* Ghanaian Cedi (GHS)
* Kenyan Shilling (KES)
* Ugandan Shilling (UGX)
* South African Rand (ZAR)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/echezona-payment-gateway-for-woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->WooCommerce->Payments screen to configure the plugin
4. Enter your Echezona API key in the plugin settings

== Frequently Asked Questions ==

= What is Echezona Payment Gateway? =

Echezona Payment Gateway is a payment processing service that allows businesses to accept payments online securely.

= Do I need an Echezona account? =

Yes, you need to sign up for an Echezona account and obtain an API key to use this plugin.

= Is this plugin compatible with WooCommerce Blocks? =

Yes, this plugin is fully compatible with WooCommerce Blocks and the new checkout experience.

= Does this plugin support High-Performance Order Storage (HPOS)? =

Yes, this plugin is compatible with WooCommerce's High-Performance Order Storage feature.
